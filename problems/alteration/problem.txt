Створiть модуль, що виконує пошук, та повертає true в разi, якщо стрічка починається зi слів `cat`, `dog`, чи `robot`, а далi слiдує число, після якого кінець речення.

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    return /.../.test(str)
  }

Ви можете скористатися оператором `|`, що означеє умову `або`.

Це оператор альтернативи:


  /xyz|qrs|tuv/.test('tuv') // true

Такий оператор можна використовувати в межах групи:

  /0x(ff|00|80)/.test('0x80') // true
