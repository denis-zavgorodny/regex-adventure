Створiть модуль, що повертає true в разi, якщо перший символ стрічки не цифра, а другий не велика лiтера.

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    // ...
  }

Для того, щоб вдало пройти цей рiвень вам потрібно навчитися використовувати заперечення в регулярних виразах.
Заперечення позначається знаком `^` на початку пiдмножини, всередині квадратних дужок `[]`.

Нагадаємо, що символ `^` за межами квадратних дужок позначає початок стрічки. Це важливо, будьте уважні.

Для того, щоб скоротити записи регулярних виразiв можна використовувати метасимволи.


  * \d - [0-9]        будь-яка цифра
  * \w - [A-Za-z_]    будь-яка лiтера
  * \s - [ \t\r\n\f]  пробіл

Великi символи позначають протележне значення:

  * \D - [^0-9]        не цифра
  * \W - [^A-Za-z_]    не лiтера
  * \S - [^ \t\r\n\f]  не пробіл

Спецiальний метасимвол `.` означає будь-який символ, за виключенням переносу стрічки.

Щоб перевiрити ваше рiшення:

  regex-adventure-ua verify solution.js
