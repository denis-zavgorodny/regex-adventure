Зараз треба створити модуль, що виконує пошук числа, яке йде за пiдстрічкою `x=`, та повертає знайдене число.

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    var m = /...your regex goes here.../.exec(str)
    return m ? m[1] : null
  }

Замiсть виклику методу `.test()` ми маємо викликати метод `.exec()`, який повертає знайдений результат. Якщо регулярний вираз спiвпадає зi стрічкою, то `.exec()` повертає match захопленої групи.

Для того щоб створити групу для захоплення в регулярному виразi ви повиннi додати на початку та кiнцi групи круглi дужки `(` та `)`.

Текст, що розташовано мiж такими дужками буде захоплено та повернено в якостi результату виконання методу `.exec()` у виглядi масиву з елементами починаючи з крайнього лiвого (якщо таких спiвпадiнь два або бiльше).
