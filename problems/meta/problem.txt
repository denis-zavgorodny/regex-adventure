Створiть модуль, що повертає true в разi, якщо передана стрічка закiнчується на `.` (крапку).

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    /* ... */
  }

`.` зазвичай, це метасимвол регулярного виразу, однак якщо перед крапкою додати зворотній слеш \ (backslash), то вона буде означати звичайну крапку (наприклад, в реченнi).
Кожного разу, коли вам потрiбно щоб метасимвол означав сам себе:

  . * + ? [ ] ( ) ^ $ { } | \

додайте перед ним `\`.

Щоб перевiрити ваше рiшення:

  regex-adventure-ua verify solution.js
