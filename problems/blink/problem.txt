Створiть аналог методу markdown, який переводить весь текст мiж `@@...@@` в текст `<blink>...</blink>`.

Наприклад, `@@whatever@@` замiняється на `<blink>whatever</blink>`.

Звернiть увагу на те, що markdown дозволяє використовувати конструкцiї `@@...@@` всерединi `@@...@@`. Ви повиннi реалiзувати таку можливiсть.

Ваш код повинен прийняти стрічку з розмiткою в синтаксисi markdown та повернути стрічку в html. Ви можете скористатися наступним шаблоном:

  var marked = require('marked')
  module.exports = function (str) {
    var md = marked(str)
    /* return html string */
  }

Для цього завдання використовуйте `marked` модуль. Щоб його встановити виконайте команду: `npm install marked`

Для того щоб виконати замiну спробуйте метод `.replace()`
