Створiть модуль, що отримує стрічку, та повертає масив зi стрічок, що були заключенi у подвiйнi лапки. Вихiдний масив повинен мiстити стрічки разом iз лапками.


Наприклад, вхiдна стрічка: 'cool "beans" "beep boop" whatever "yay"'
ваша програма повинна повернути масив: ['"beans"', '"beep boop"', '"yay"']

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    return str.match(/.../g)
  }

Ви повиннi використовувати `str.match(regex)` щоб повернути массив для `regex` з флагом `/g`.

Вам можуть знадобитися заперечення в регулярних виразах.

Щоб перевiрити ваше рiшення:
