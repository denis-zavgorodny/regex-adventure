Стрічки в javascript мають метод `.split()`. Цей метод роздiляє стрічку на частини, та результат повертає у виглядi масиву. Цей метод може приймати в якостi роздiлювача стрічку або регулярний вираз.

Сворiть модуль, що приймає стрічку, та повертає масив, елементами якого є частини стрічки, що були роздiленi символами ` , `(кома з пробiлами на кiнцях)

Ви можете скористатися наступним шаблоном:

  module.exports = function (str) {
    return str.split(/* ... your regex goes here ... */)
  }

Нагадаємо, що `\s` – це скорочення для символу пробiлу `[ \t\r\n\f]`.
